Bale Counting

Our bale counting system uses computer vision and geometric calibration to automatically calculate the number of bales on a trailer, no human input required.

By calibrating a single camera to the dimensions of the trailer, the system “flattens” the image so each bale appears in perfect alignment, allowing rows and columns to be detected with precision. Unlike AI models that rely solely on object recognition (which often struggle with straps, lighting, or shadows), our approach is based on maths, not guesswork.

Once set up, it runs continuously, tracking each trailer as it arrives and calculating the bale count in real time. This saves hours during harvest and removes the risk of human error when recording loads. Designed to run on affordable devices like NVIDIA Jetson modules, the system brings advanced automation to farms of all sizes without the need for expensive infrastructure.

Weighbridge Automation

The weighbridge automation system removes one of the slowest parts of the farm logistics chain manual data entry.

Using camera feeds positioned at the weighbridge, the system automatically detects when a vehicle enters or leaves, captures images, and identifies trailers and load types. Each event is logged, timestamped, and uploaded to a central dashboard, allowing farm managers to track deliveries and materials remotely.

Because it runs directly on-site using edge devices, it works even with limited internet connectivity and integrates easily with existing weighbridge setups. This means drivers no longer need to exit the cab, and the weighing process becomes faster, safer, and more consistent. Over time, the system builds a digital record of every load creating reliable traceability and operational insights.
